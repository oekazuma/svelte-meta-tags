| プロパティ | 型                          | 説明                                                                                                                 |
| ---------- | --------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `output`   | string (デフォルト値: head) | json-ld を `<head>` または `<body>` のどちらに出力するかを指定します。可能な値は `head` または `body` のいずれかです |
| `schema`   | Object                      | `ld+json` 形式のデータ                                                                                               |
